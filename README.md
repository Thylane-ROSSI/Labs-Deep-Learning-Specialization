## Course Overview

### [1. Neural Networks and Deep Learning](./1-Neural-Networks-and-Deep-Learning)
Focuses on the foundational concepts of Deep Learning.
* **Key Labs:** Logistic Regression with a Neural Network mindset, Planar data classification with a hidden layer, Building a Deep Neural Network (Step-by-Step), and Cat vs. Non-Cat Image Classification.
* **Skills:** Python, NumPy, Vectorization, Forward & Backward Propagation.

### [2. Improving Deep Neural Networks](./2-Improving-Deep-Neural-Networks)
Techniques to open the "black box" and systematically improve model performance.
* **Key Topics:** Initialization, L2 and Dropout regularization, Hyperparameter tuning, Batch Normalization, and Optimization (Adam, RMSprop, Momentum).
* **Framework:** TensorFlow.

### [3. Structuring Machine Learning Projects](./3-Structuring-Machine-Learning-Projects)
Strategic decision-making for ML project leaders.
* **Key Topics:** Error Analysis, Mismatched training/test sets, Transfer Learning, and Multi-task learning.

### [4. Convolutional Neural Networks](./4-Convolutional-Neural-Networks)
Foundations of Computer Vision.
* **Key Topics:** CNN architectures (ResNets, Inception), Object Detection, Face Recognition, and Neural Style Transfer.

### [5. Sequence Models](./5-Sequence-Models)
Building models for natural language and sequential data.
* **Key Topics:** RNNs, LSTMs, GRUs, Word Embeddings, Hugging Face Tokenizers, and Transformers.
