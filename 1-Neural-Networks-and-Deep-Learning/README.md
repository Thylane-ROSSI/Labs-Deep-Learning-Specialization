# Course 1: Neural Networks and Deep Learning

This directory contains the core programming assignments for the first course of the specialization. These labs cover the implementation of neural networks from scratch using NumPy.

## Labs Overview

### 1. [Logistic Regression with a Neural Network mindset](./Lab%201%20-%20Logistic_Regression_with_a_Neural_Network_mindset.ipynb)
* **Objective:** Build a logistic regression classifier to recognize cats in images.
* **Key Concepts:** Parameter initialization, cost function calculation, and gradient descent optimization.

### 2. [Planar data classification with one hidden layer](./Lab%202%20-%20Planar_data_classification_with_one_hidden_layer.ipynb)
* **Objective:** Build a neural network with a single hidden layer to classify non-linearly separable data.
* **Key Concepts:** Implementing tanh activation functions and computing cross-entropy loss.

### 3. [Building your Deep Neural Network: Step by Step](./Lab%203%20-%20Building_your_Deep_Neural_Network_Step_by_Step.ipynb)
* **Objective:** Modular implementation of all the functions required to build a deep (L-layer) neural network.
* **Key Concepts:** ReLU activation, deep layer initialization, and general forward/backward propagation blocks.

### 4. [Deep Neural Network - Application](./Lab%204%20-%20Deep%20Neural%20Network%20-%20Application.ipynb)
* **Objective:** Final application of the course, building and training an L-layer deep network for cat vs. non-cat classification.
* **Key Concepts:** Fine-tuning hyperparameters and observing the performance gain of deep architectures over logistic regression.
